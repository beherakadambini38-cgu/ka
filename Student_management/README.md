#  Student Management System 

##  Project Description

This project is a simple **Student Management System** developed using Java.
It helps manage student records such as **name, roll number, and grade** using a menu-driven program.

The system allows users to add, view, and search student details easily.

---

##  Features

* Add new student
* Display all students
* Search student by roll number
* Save data to file
* Load data from file
* Simple and easy menu interface

---

##  Technologies Used

* Java
* ArrayList
* File Handling (ObjectInputStream / ObjectOutputStream)
* Programiz Online Compiler

---

##  How to Run the Project

1. Open the project link:
    https://www.programiz.com/online-compiler/3O0eCi6xnKDbK

2. Click on **Run **

3. Choose options from menu (Add, Display, Search, Exit)

4. Enter student details when asked

Programiz provides an online compiler where you can write and run Java code directly in your browser without installing anything. ([Programiz][1])
